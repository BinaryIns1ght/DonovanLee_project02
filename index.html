Project Overview

In this project, the main objective is to utilize p5.js's user interactivity. Though the events and manipulation of variable values, this project is meant to allow the user to interact with the website through using the sliders, buttons, and color picker. The inspiration of this project is mostly based around the "Mad Libs" book series, along with the many other site that allow customization of the user's size.

Instructions

  • Use the Sliders To Change Background Color
  • Replace the default words with your custom words to chagne the story
  • Press the "Get A Random Score" button to get a randomly generatred score from 70-100
  • Click On The Color Pickers to choose a color for the text and the text background
  • Drag the shapes around the project to add more life to the design.
  • You can left and right click on the mouse and see how the circle reacts to it.

Prototyping

When it comes to making a prototype for this project, there was mostly experimentation around the simpler aspects of the project, like the color sliders, or the color picker. There has been many struggles around placement of the elements and how to work certain aspects of each element. Many more options on interactivity would be later added here.
This was an early testing phase in the p5js editor, where the elements of the project's concepts are being tested on their capabilities and how much more each element can change up the webpage.
The code was made from the "ground-up". With the help with many sources of code provided by the official p5 site.

Testing

Each feature shown here was both tested separately and together. They're separately tested to ensure their function are working properly. Having them tested out together allows an understand how each element works with each other.

Final Design

When making the final design for this project the purpose was to add many appropriate changes to the site, adding more options to the design. This final iteration now allows the users to drag and move around the shapes at the lower half of the project, while also making a new narrative with them. The user also now has the ability to change the background color of the passage the user is creating, and also allows the user to view the specifics of the colors, like Hex Codes and RGBA Codes.
